# Planejy
O Planejy é um projeto criado com o objetivo de ajudar as pessoas que possuem dificuldade em planejar seu dia ou que são afetadas por procrastinarem a realização de tarefas.
As aplicações foram criadas pensando justamente em ser algo rápido e fácil de utilizar mas que contemple todas as funcionalidades necessárias para um bom planejamento das tarefas.
